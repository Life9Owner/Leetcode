Contest Problem
Time Limit：C/C++ 1000MS，Other 2000MS
Memory Limit：C/C++ 256MB，Other 512MB
Score：100
Description

Input

第一行输入两个整数 n,m,x,y ，分别代表地图的长度、宽度、小闽可以移动的最大步数，小涛可以移动的最大步数。

Output

输出两个整数，分别代表小闽和小涛可能找到的猫咪数量以及所需的最小移动步数之和。
